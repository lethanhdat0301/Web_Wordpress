# Sing Trade - Wordpress Project ## Yêu cầu: * **XAMPP** * **PHP** * **MySQL/MariaDB**. ## Cấu hình Database: * Khởi động Apache và MySQL trong XAMPP Control Panel. * Truy cập http://localhost/phpmyadmin. * Tạo một Database mới (Ví dụ đặt tên là: wp_project_db). * Chọn Database vừa tạo, nhấn vào tab Import. * Chọn file .sql có trong thư mục code vừa tải về. * Nhấn Go để import dữ liệu. ## Kết nối Database Chỉnh sửa file wp-config.php để phù hợp với máy cá nhân. ``` text define( 'DB_NAME', 'wp_project_db' ); define( 'DB_USER', 'root' ); define( 'DB_PASSWORD', '' ); ``` ### Cập nhật đường dẫn: ``` text define( 'WP_HOME', 'http://localhost/ten-thu-muc-du-an' ); define( 'WP_SITEURL', 'http://localhost/ten-thu-muc-du-an' ); ``` ### Truy cập trình duyệt: * Truy cập trình duyệt: http://localhost/ten-thu-muc-du-an. * Đăng nhập vào trang quản trị: http://localhost/ten-thu-muc-du-an/wp-admin.
